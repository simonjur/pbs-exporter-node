# Security Policy

## Reporting a Vulnerability

Security bugs in this project are taken seriously. Thank you for responsibly
disclosing your findings.

Please report security issues privately via the GitHub Security Advisory
["Report a Vulnerability"](https://github.com/simonjur/pbs-exporter-node/security/advisories/new)
tab for this repository. Do **not** open a public issue for security problems.

You will receive a response indicating the next steps in handling your report.
We will keep you informed of the progress toward a fix and may ask for
additional information or guidance.
